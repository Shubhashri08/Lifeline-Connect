# Lifeline Connect

**Empowering Rural Healthcare Access in India**

Lifeline Connect is a comprehensive digital health platform designed to bridge the gap between rural patients and healthcare facilities. It enables users to find nearby hospitals, check real-time availability, and book secure appointments effortlessly.

## Getting Started

1.  Clone the repository.
2.  Navigate to the `frontend` directory: `cd frontend`
3.  Install dependencies: `npm install`
4.  Run the development server: `npm run dev`
5.  Open [http://localhost:3000](http://localhost:3000) in your browser.
